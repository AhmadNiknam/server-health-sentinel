# Screenshot Guidance

Do not upload screenshots containing real machine names, real server names, IP addresses, subscription IDs, tenant IDs, usernames, credentials, tokens, management endpoint addresses, or operational details.

Use redacted screenshots only. Blur, crop, or replace sensitive values before adding images to the repository or sharing them publicly.

Recommended screenshots:

- HTML report executive summary
- Hybrid mode console output with sensitive values blurred
- GitHub Actions passing
- Project folder structure
- README preview

Generated reports may contain real operational data. Do not commit report files just to provide screenshots.
