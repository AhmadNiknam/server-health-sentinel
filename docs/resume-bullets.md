# Resume Bullets

## System Administrator

- Built a read-only PowerShell health reporting tool for Windows Server environments, generating HTML, CSV, and JSON reports for pre-maintenance and weekly review workflows.
- Automated checks for storage, network adapters, event log risk indicators, pending reboot signals, critical services, and maintenance readiness.
- Designed sample-based configuration and Git ignore rules to keep real server names, reports, history snapshots, and sensitive operational details out of source control.
- Added Pester tests and GitHub Actions CI to validate script behavior and support repeatable project quality.

## Network Analyst

- Developed network adapter health checks and Hybrid reporting workflows to identify link state, adapter visibility, and connectivity-related findings across server inventories.
- Created admin-friendly reports that summarize unreachable targets, remote management gaps, and network-related readiness issues for operational review.
- Documented safe demo scenarios and public repository practices to avoid exposing real hostnames, IP addresses, or operational data.

## Cloud / Azure Administrator

- Implemented Azure VM health reporting design using Az PowerShell concepts, VM metadata checks, subscription/context validation, and guest readiness findings.
- Built Hybrid mode to combine local, on-prem, and Azure VM health signals into consolidated reports for cross-environment review.
- Documented Azure permission considerations, VM Run Command requirements, and safe handling of sample Azure inventory without exposing tenant or subscription details.

## IT Automation / PowerShell

- Engineered a modular PowerShell reporting tool with separate configuration loading, collector modules, health evaluation, predictive risk indicators, trend history, and report generation.
- Added rule-based predictive risk and trend comparison features to highlight worsening, stable, improving, or unknown operational indicators without claiming exact failure prediction.
- Created optional hardware sensor readiness design for future Redfish/iDRAC/iLO integration while preserving read-only execution and secure credential boundaries.
- Produced professional GitHub documentation, demo scenarios, security notes, limitations, roadmap, changelog, and portfolio-ready project materials.
