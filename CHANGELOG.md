# Changelog

## Version 0.1.0 - Initial Functional Release

- Added project structure for a modular PowerShell health reporting tool.
- Added sample-based configuration loading.
- Added local Windows health checks.
- Added on-prem Windows Server health checks.
- Added Azure VM health checks and readiness findings.
- Added Hybrid mode across local, on-prem, and Azure inventory.
- Added professional HTML, CSV, and JSON reports.
- Added trend history snapshots and comparisons.
- Added predictive risk indicators using rule-based early warning signals.
- Added optional hardware sensor readiness.
- Added Pester tests.
- Added GitHub Actions CI.
